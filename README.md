# De Patria a Patria

## Descripción

**De Patria a Patria** es un proyecto web responsivo que presenta la Galería de Arte TripleTen. El sitio muestra un viaje épico de Kentucky a Burundi, pasando por Gales y Ucrania, celebrando las historias y lugares de origen de los profesionales que forman parte de la comunidad TripleTen.

## Funcionalidad

- Diseño responsivo que se adapta a tres resoluciones principales:
  - **Desktop:** 1280px
  - **Tablet:** 768px
  - **Mobile:** 320px

## Tecnologías y técnicas utilizadas

- **HTML5** - Estructura semántica del contenido
- **CSS3** - Estilos y diseño visual
  - Flexbox para layouts
  - Media Queries para diseño responsivo
  - @font-face para fuentes personalizadas
- **Metodología BEM** - Organización de clases CSS
- **Estructura de archivos BEM** - Organización modular de bloques CSS
- **Normalize.css** - Reset de estilos para consistencia entre navegadores
- **Diseño responsivo** - Puntos de ruptura en 544px y 1024px
